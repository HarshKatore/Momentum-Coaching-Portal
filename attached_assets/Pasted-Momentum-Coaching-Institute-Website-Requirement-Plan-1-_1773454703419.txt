2. Technical Requirements (Strict)

The website must follow these constraints.

Framework

Must use

React JS

Recommended setup

React + Vite

No backend allowed.

Data Handling

All dynamic data must come from editable JSON files.

Example:

src/data/
courses.json
faculty.json
toppers.json
branches.json
gallery.json
testimonials.json

This allows non-developers to edit content easily.

Global Config System

All keys, contact info, links must be editable from one config file.

Example

src/config/siteConfig.js

Example structure

export const CONTACT = {
  phone: "+91XXXXXXXXXX",
  whatsapp: "919XXXXXXXXX",
  email: "info@momentumcoaching.in"
}

export const SOCIAL = {
  youtube: "",
  instagram: "",
  telegram: ""
}

export const SITE = {
  instituteName: "Momentum Coaching Institute",
  tagline: "Building Future IITians & Doctors",
  foundedYear: "2025"
}

Agency must not hardcode these values anywhere else.

Environment Variables

Example .env

VITE_GOOGLE_MAP_URL=
VITE_SITE_DOMAIN=

Secrets must not be committed to GitHub.
